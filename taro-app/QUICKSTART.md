# 快速开始

## ✅ 构建成功！

项目已成功构建，`dist` 目录已生成。

## 下一步：在微信开发者工具中测试

### 1. 打开微信开发者工具

如果还没有安装，请先下载：
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目

1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择项目目录：`/Volumes/geek/bb-2025/delicious-moments/taro-app/dist`
4. AppID：选择「测试号」或输入你的小程序 AppID
5. 项目名称：食光集

### 3. 开发模式（推荐）

在终端运行开发模式，支持热更新：

```bash
cd taro-app
npm run dev:weapp
```

然后在微信开发者工具中点击「编译」即可看到实时更新。

## 已知问题

1. **TabBar 无图标**：为了快速运行，暂时移除了图标配置，只显示文字
   - 如需添加图标，准备 8 张图标文件（81x81px）放到 `src/assets/icons/`
   - 然后在 `src/app.config.ts` 中配置 iconPath 和 selectedIconPath

2. **SCSS 警告**：Sass @import 弃用警告，不影响功能，后续可以改用 @use
